# EEE4113F_Group12_PCB
## Overview
This repository was used for Group 12 (of the 2026 EEE4113F course from the University of Cape Town) to collaborate on the design of the project's PCB. This PCB was eventually sent to [JLCPCB](https://jlcpcb.com/) for printing.

## What is included
This repository includes:
- The Project file
- The Schematic file
- The PCB file

None of the generated gerber files are included.

## Installation & Dependencies
The files were made with KiCAD 9.0, so the same or a later version of KiCAD is needed to open the files.
